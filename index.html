WELCOME

Are you good at python? Are you bad at python? Doesn't matter, I'm making this to get a web developer certification so enjoy. Here are some reasons that python is a cool language:

  • easy to learn
  • free!
  • often used in machine learning
  • lots of convenient APIs to import and use
  • abundance of resources to look up in case you get lost (like this one)

So what are you waiting for? Install your favourite locally sourced free-range python compiler and get started!
